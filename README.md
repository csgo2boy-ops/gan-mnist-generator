# GAN 生成手写数字 - 我的第一个AI项目

用 PyTorch 从零实现 GAN（生成对抗网络），从随机噪声生成逼真的手写数字（MNIST 数据集）。

**自学AI，不需要大学文凭！**  
我高中辍学后，通过AI导师一步步学习，亲手跑通了这个项目。行动比背景重要！

## 项目亮点
- 从纯噪声开始，训练50轮后，生成器能创建清晰可辨认的手写数字（1、5、7、4、9、3、6、8 等）
- 看到AI“学会写字”的完整过程，超级有成就感！
- 代码简单易懂，适合新手学习GAN原理
